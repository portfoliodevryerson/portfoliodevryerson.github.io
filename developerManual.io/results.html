Here are our results

Front End: Javascript

Rating: 8/10; tedious to work with but generates amazing results on the client side. This was not difficult to install but there's a learning curve to the language.

There were lots of Javascript libraries used like bootstrap, jquery code, carousel views, magnific-popup and sticky.

  • 8/10
  • Would use again
  • Rendered great front end

Back End: PHP

9/10; PHP is an amazing scripting language and allows me to run server side code easily.

  • Very easy to use on databases. Bluehost and a variety of other servers support PHP.
  • Would use again.

Front End: CSS

5/10; difficult to pick up and style sheets required lots of editing

This requires more effort from my part, and I've done these layouts in the past which is why they were easy to change. Without previous modifications, it would've been hard for me to change the stylesheets.

  • Rendered great content via HTML sheets
  • Extremely tedious development

Front End: HTML

10/10; very easy to pickup and easily can integrate code into different projects.

Variety of tags made it very easy to style

One downside is having to manage a variety of pages - we couldn't do this dynamically.

  • Text editor is require with no complex IDEs
  • Easy to use and learn
  • Variety of libraries makes it easy to develop
