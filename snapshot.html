Here is our snapshot summary

Front End: Javascript

Popularity: According to the Stack Overflow 2018 developer survey this technology is the most popular language to work with since it was chosen by 69.8% of voters.

Disadvantage: The code executes on the usersâ€™ computer, in some cases it can be exploited for malicious purposes.

Disadvantage: Javascript can be interpretted differently by different browsers.

  • Very fast and can be run immediately via the client side browser.
  • Fairly simple to learn and implement
  • Can be inserted into various applications regardless of their extension.
  • Client side rendering reduces demand on website server.

Back End: PHP

Popularity: According to the Stack Overflow 2018 developer survey this technology was voted most popular by 30.9% of voters.

Disadvantage: Servers might slow down if many PHP scripts are run since the client side cannot be executed.

  • Supports database connectivity. PHP can access over 20 different databases including MySql, Oracle, and MS Access.
  • Supports sessions and PHP can generate unique session IDs. The ID follows the user during a single session on a web site.
  • Reduces development time because PHP can easily be picked up.
  • Maintains source code security.

Front End: CSS

Popularity: According to the Stack Overflow 2018 developer survey this technology was voted most popular by 30.9% of voters.

Disadvantage: Operates differently on various browsers (i.e IE and Opera support different stylesheets)

  • Aesthetically pleasing layouts, designs and formatting
  • Reduces file transfer size
  • Allows for dynamic layouts on various platforms
  • Separates document content
  • Simplifies HTML pages

Front End: HTML

Popularity: According to the Stack Overflow 2018 developer survey this technology was the second most popular language to work with since it was chosen by 68.5% of voters.

Disadvantage: Can only create static pages, nothing dynamic and no state management.

Disadvantage: Not very responsive and divs are easier to work with.

Disadvantage: Lots of time is spent on developing a web page from scratch via HTML

  • Widely used on all platforms
  • All browsers support the technology
  • Easy to use and learn
  • Plain text is easy to edit
  • Can present any type of data
